1. ResizeObserver:
    ResizeObserver is a browser API that lets you watch an element and get notified whenever
    its size (width/height) changes.

    e.g:
      useEffect(() => {
        const element = document.querySelector('#my-element');
        const boxes = document.querySelectorAll('.box');

        const observer = new ResizeObserver((entries) => {
          for (const entry of entries) {
            console.log('Resized:', entry.target, entry.contentRect);
          }
        });

        // observe one element
        observer.observe(element);

        // observe multiple elements
        boxes.forEach(box => observer.observe(box));

        return () => {
          // stop observing one element
          observer.unobserve(element);

          // for un-observing everything at once
          observer.disconnect();
        }
      })
  
      - entry.target: the observed element
      - entry.contentRect: an object with .width, .height, etc.
      - entry.borderBoxSize, entry.contentBoxSize: more detailed box info (useful for complex cases)

      ## using useRef
      -
      const containerRef = useRef<HTMLDivElement | null>(null);
      const [dimensions, setDimensions] = useState({ width: 500, height: 300 });

      useEffect(() => {
        const observer = new ResizeObserver((entries) => {
          const entry = entries[0];
          const { width } = entry.contentRect;
          setDimensions({ width, height: 300 });
        });

        if (containerRef.current) {
          observer.observe(containerRef.current);
        }

        return () => observer.disconnect();
      }, []);

2. window.dispatchEvent:
  # Description:
    window.dispatchEvent is a JavaScript method that lets you manually fire (dispatch) an event on the window object,
    triggering any event listeners attached to it. Unlike browser-generated events, it runs synchronously and
    is often used to simulate user actions or custom events.

  # Key details:
    1. Event Creation: You must first create an event object using the Event constructor (or specialized constructors like MouseEvent, KeyboardEvent, etc.).
      Example:
        let event = new Event('resize');
        window.dispatchEvent(event);

    2. Event Authenticity: Programmatically dispatched events have event.isTrusted = false, distinguishing them from user-generated events (true)

  E.g: Simulate resize:
    window.addEventListener('resize', () => console.log('Resized!'));
    window.dispatchEvent(new Event('resize'));
    // Logs "Resized!" without actually resizing the window.

  E.g: Trigger custom events:
    let myEvent = new Event('myCustomEvent', {bubbles: true, cancelable: true});
    window.dispatchEvent(myEvent);
    // Testing event listeners: Useful in unit tests to verify event-driven code without requiring user interaction.

  E.g: Using it in React
    const [theme, setThemeState] = useState<ThemeMode>('auto')

    useEffect(() => {
      const handleThemeSync = () => {
        setThemeState(getInitialMode())
      }

      window.addEventListener('theme-change', handleThemeSync)
      return () => {
        window.removeEventListener('theme-change', handleThemeSync)
      }
    }, [])

    const setTheme = (newTheme: ThemeMode) => {
      setThemeState(newTheme)
      applyThemeMode(newTheme)
      window.localStorage.setItem('theme', newTheme)
      window.dispatchEvent(new Event('theme-change'))
    }

    return { theme, setTheme }

    - From the example above, you'll see that the "useEffect" runs and subscribes to the customEvent 'theme-change',
      and in "setTheme" function, we fire the custom event 'theme-change', so every callback subscribed to 'theme-change'
      will be fired when the "dispatchEvent" is triggered

  E.g Passing data with events:
    caller:
      window.dispatchEvent(
        new CustomEvent('theme-change', {
          detail: {theme: 'dark'}
        })
      )
    
    Listener:
      window.addEventListener('theme-change', (event: CustomEvent<{theme: string}>) => {
          console.log(event.detail.theme)
        }
      )
